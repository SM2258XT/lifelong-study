1.  Nacos是什么？
一个更易于构建云原生应用的动态服务发现、配置管理和服务管理平台。

Nacos：Dynamic Naming and Configuration Service

Nacos就是注册中心＋配置中心的组合 -\> Nacos = Eureka+Config+Bus
1.  Nacos能干什么？
替代Eureka做服务注册中心，就不用再去创建项目配置eureka注册中心了！
