\# Fock/Join：分支合并

2022年9月10日

**Fock/Join概念：**把大任务分割成若干个小任务，最终汇总每个小任务结果后得到大任务结果。其实现思想与MapReduce有异曲同工之妙。

例：计算1+2+3+…+9+100，要求每次相加的两个数字只差，不超过10。
