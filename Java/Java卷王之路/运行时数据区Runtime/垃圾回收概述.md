1.  什么是垃圾？
垃圾是指在运行程序中没有任何指针指向的对象，这个对象就是需要被回收的垃圾。
1.  为什么需要GC？
除了释放没用的对象，垃圾回收也可以清除内存里的记录碎片。碎片整理将所占用的堆内存移到堆的一端，以便JVM将整理出的内存分配给新的对象。
1.  回收哪里？
Java**堆**是垃圾收集器的工作重点

从次数上讲：
1.  频繁收集Young区
2.  较少收集Old区
3.  基本不收集Perm区（元空间）
